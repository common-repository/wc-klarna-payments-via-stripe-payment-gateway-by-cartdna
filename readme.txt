== Description ==

= Integrate Klarna Payments by Stripe with WooCommerce Wordpress store =  

This Plugin adds Klarna payments by stripe as a payment gateway for WooCommerce.
You must have an active Stripe account with enabled Klarna payments to use our plugin.
It uses the redirect method. SSL is not mandatory.

= Main features =

* All Klarna Payment options available (Klarna Pay Now, Klarna Pay Later, Klarna Slice it) based on country available payment option can differ.
